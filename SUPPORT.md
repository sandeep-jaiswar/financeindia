# Support for financeindia

Thank you for using `financeindia`! We want to ensure you have a great experience. Here is how you can get support:

## 1. Check the Documentation
Before opening an issue, please review the [README.md](README.md) and any relevant docstrings in the code. Most common usage questions are covered there.

## 2. GitHub Issues
If you've found a bug or have a feature request:
- **Bugs**: Open an issue on our [GitHub Repo](https://github.com/sandeep-jaiswar/financeindia/issues). Please include a minimal reproducible example (code snippet) and your OS/Python version.
- **Feature Requests**: We welcome ideas! Open an issue with the label `enhancement` and describe your use case.

## 3. Discussions
For general questions, "How do I...?", or brainstorming:
- Use the **GitHub Discussions** tab on our repository.

## 4. Security Issues
If you have found a security vulnerability, please refer to our [Security Policy](SECURITY.md). **Do not open a public issue.**

## 5. Commercial Support
If you require priority support, consulting, or custom feature development for your organization, please contact [jaiswarsandeep119@gmail.com](mailto:jaiswarsandeep119@gmail.com).

---
*Please note: This is an open-source project maintained by volunteers. We will do our best to respond, but cannot guarantee immediate support.*
